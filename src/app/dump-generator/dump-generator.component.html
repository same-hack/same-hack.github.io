【PostgreSQL】dumpファイルジェネレーター

スキーマ単位でのdumpファイル生成ツールだよ！
コマンドプロンプトに入力してね！


pg_dump -h {{ host }} -p {{ port }} -U {{ user }} -d {{ db }} -n {{ schema }} -F c -f {{ nameFile }}.dump

[ オプションについて ]
-h　 : ホスト
-p 　: ポート番号※デフォルトは5432
-U 　: ユーザー名
-d 　: データベース名
-n 　: スキーマ名
-F c : ダンプ形式の指定。c はカスタム形式を意味します。他にも t（テキスト形式）、d（ディレクトリ形式）などがあります。
-f 　: ダンプデータを保存するファイルの指定

【PostgreSQL】リストアコマンド

上で作ったdumpファイルのリストアコマンドだよ！
ダンプファイルを別の場所に保管した場合は、そのファイルのパスを指定してね！
コマンドプロンプトに入力してね！

pg_restore -h {{ host }} -p {{ port }} -U {{ user }} -d {{ db }} -v {{ nameFile }}.dump
